I'm glad to announce that version 0.3.0 of YajHFC has been released today.

Download:
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_0.jar
Source code:
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_0-src.zip

New features:
- Support to convert pictures to PostScript before submitting a fax job
- A "Preview" button to the Send dialog
- Support for phone books in SQL databases (using JDBC)
- Some internal changes

For more information you may also take a look at the homepage at:
http://yajhfc.berlios.de/


Currently BerliOS seems to have some problems with the site's availability, so 
if the homepage and/or download doesn't work, please check back a few hours 
later.

Have fun,
Jonas
