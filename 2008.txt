I am pleased to announce that version 0.3.6 of YajHFC has been released today.

Download:
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_6.jar

Source code:
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_6-src.zip

Windows setup.exe:
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_6-setup.exe

New features/changes:
- Some bug fixes
- Support for selection of multiple phone book entries for the send dialog
- New command line argument ("--logfile") to log to a file in debug mode
- Support for "disconnected mode" as a possible workaround to some HylaFAX 
bugs
- Support to select the modem to use

For more information you may also take a look at the homepage at:
http://yajhfc.berlios.de/

Have fun,
Jonas

I am pleased to announce that version 0.3.7 of YajHFC has been released today.

Download: 
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_7.jar
Source code: 
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_7-src.zip
Windows setup.exe: 
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_7-setup.exe

New features/changes:
- New simplified send dialog
- Support for cover pages in HTML format
- Support for cover pages in XSL:FO and ODT format using a plugin
- Better plugin support
- Logging changed to use the Java logging APIs
- Now uses gnu.hylafax 1.0.1
- Some bug fixes

As mentioned above, YajHFC now supports cover pages in XSL:FO and OpenDocument 
Text format. Because the required libraries are rather large (11MB), a 
separate distribution/plugin including this support has been created.

You can download this plugin from the following locations:
General distribution (cross-platform, already includes YajHFC):
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_7-FOPPlugin-0_1.zip
Source code:
http://prdownload.berlios.de/yajhfc/FOPPlugin-0_1-src.zip
Windows setup.exe (already includes YajHFC):
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_7-FOPPlugin-0_1-setup.exe

For more information you may also take a look at the homepage at:
http://yajhfc.berlios.de/

Have fun,
Jonas

The Windows setup program has been updated to support installing a fax printer 
for 64bit versions of Windows, too.

You can downloaded an updated setup.exe at: 
http://prdownload.berlios.de/yajhfc/yajhfc-0_3_8b-setup.exe

This file should work with both 32bit and 64bit versions of Windows.
If you find any problems with the updated setup (with 32bit or 64bit Windows), 
please tell me.

Please note that you will need to have a 64bit version of Java installed if 
you use a 64bit version of Windows.
